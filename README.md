# Movie Recommendation System

A content-based Movie Recommendation System built using Python, Machine Learning, Streamlit, and TMDB API.

## Features

- Recommends top 5 similar movies based on user selection.
- Uses genres, cast, crew, keywords, and movie overview for recommendations.
- Displays movie posters using TMDB API.
- Interactive user interface built with Streamlit.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- TMDB API

## Dataset

- TMDB 5000 Movies Dataset
- TMDB 5000 Credits Dataset
